# npm prune

An npm version of [xthezealot/npmprune][repo].

## Usage

```bash
# use once
npx npmprune

# ensure you're using the latest version
npx npmprune@latest

# or install it globally, and then you can use it anywhere
npm i -g npmprune
npmprune
```
